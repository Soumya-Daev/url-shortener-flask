This web app is basically a url shortener like bit.ly, where you can provide large urls and it return you easy to remember, short versions of them.
This web app is made with HTML, CSS, vanilla JS (no external library),
python's flask library and sqlite3 database to store the shortened urls and their resp. original forms. No API is used.
